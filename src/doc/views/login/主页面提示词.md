# HomePage页面
## 需求
**项目需求**
你是一个前端工程师，使用vue+ts完成老鱼人个人知识库的主页面

**页面结构**
总共两个部分，顶部topbar与topbar下方的main块，他们宽都是视口的100%
1. topbar高度不用做太高，topbar右边是带下划线的表项，有GitHub仓库，文档等选项
2. topbar不需要太显眼，颜色可以使用淡色，相较于下方的main块
3. main块主内容在块正中心，由上往下分三个部分，大标题（老鱼人知识库），搭建知识库的一小句推广词，对话和搭建知识库按钮（左右间距适中，但要比大标题窄）
4. 整个页面采用蓝白色调，要有聚焦中心（main的主内容块）的效果
5. 按钮需要浮动效果，大标题需要渐变色，最好在main块的其它地方能有粒子

**核心逻辑**
暂不需要实现ts的核心逻辑，完成静态页面即可
